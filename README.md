# React Cards Page

## Initial structure
```
<Header/>
<CardsPage>
    <CardsFilter />
    <CardPreview />
    <CardsList>
        <Cards/>
        ...
    </CardsList>
</CardsPage>
<Footer/>
```

## Running locally

```
#NPM to install dependencies
npm install 

#Running in webpack dev server
npm start
```

Dev server available [here](http://localhost:3000/webpack-dev-server)
